# 初期化処理

## 概要

* ボットの起動時に以下の処理を実行する
  1. コマンドライン引数の取得
  2. ボットクライアントのインスタンスを生成する
  3. ボット設定ファイルの読み込み
  4. トークンを使用してボットを起動
  5. DBとDiscordサーバーのチャンネル情報を同期する
* 1~4までの処理はindex.jsで実行する
* 5はready.jsで実行する
  * ※ログイン開始時点ではチャンネル情報を取得できないため
